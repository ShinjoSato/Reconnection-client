## 概要

チャット履歴やユーザー情報などの個人情報を自らが管理できるチャットアプリが欲しくて開発を始めました。

## 機能

- **リアルタイムチャット**

    送受信したデータが自分または相手のUIにリアルタイムで反映されます。

## 仕様技術

React, Material UI, SocketIO
